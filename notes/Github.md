``` 
git --version

### Set Username
```
git config --global user.name amexias
```
### Set email
```
git config --global user.email amexias@gmail.com
```
### Create Repository
```
git init 
```
### Status
```
git status
```
### Add File
```
git add index.html
```
### Remove file
```
git rm --cached index.html
